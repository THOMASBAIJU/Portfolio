# Thomas Baiju | Portfolio

Welcome to my personal portfolio repository! This project showcases my skills as a Full-Stack & AI Engineer.

## Tech Stack
- **Frontend:** React (Vite) + Tailwind CSS + Framer Motion
- **UI/UX:** Custom premium dark aesthetic with glassmorphism effects
- **Icons:** Lucide React

## Running Locally

Clone the project and run the following commands in the project directory:

```bash
# Install dependencies
npm install

# Start the development server
npm run dev
```
